¿Que es Python?

Python es un lenguaje de programación interpretado, interactivo y orientado a objetos. Incorpora módulos, excepciones, tipado dinámico, tipos de datos dinámicos de muy alto nivel, y clases.

Python combina potencia con una sintaxis muy clara. Tiene interfaces a muchas llamadas al sistema y bibliotecas, así como también a varios sistemas de ventanas, y es extensible en C o C++.  También es utilizable como un lenguaje de extensión para aplicaciones que necesiten interfaces programables. Finalmente, Python es portable, corre en muchas variantes de Unix, en la Mac, y en PCs bajo MSDOS, Windows, Windows NT, y OS/2.
